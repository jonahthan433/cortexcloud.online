Dynamic Routing
Route tasks to agents based on capability, load, or model — at runtime. Conditional branching with fallback paths.

Retry & Recovery
Per-step retry policies with exponential backoff. Automatic checkpoint-resume on failure. Zero task loss.

Parallel Execution

Define workflows as directed acyclic graphs. Version-controlled. Exportable as YAML. GitOps compatible.

Enterprise-grade control on every action

Docker / SSH / native execution modes. Seccomp syscall hardening. Linux namespace isolation per agent. Escape-proof by design.

04 — Scale

From a laptop to a GPU cluster

One platform, three deployment modes. Start local, scale horizontally, or go fully air-gapped — without changing your agent code.

🖥️

Single Node

Ollama on CPU or GPU, SQLite, and a single docker compose up. Perfect for solo builders and small teams who want to move fast without infrastructure overhead.

▸ Runtime: Ollama (CPU/GPU)
▸ Storage: SQLite + local filesystem
▸ Deploy: docker-compose.yml
▸ Ideal: Teams of 1–10
